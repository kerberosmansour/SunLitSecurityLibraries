# `docs/slo/completed/` - finished runbooks

Runbooks here are closed and retained as durable project history. Completion
summaries live in [`../completion/`](../completion/), and lessons learned live
in [`../lessons/`](../lessons/).

These files may describe the repository state at the time a runbook closed. For
current licensing, security, and contribution policy, use the root-level
`LICENSE`, `SECURITY.md`, and `CONTRIBUTING.md` files.
