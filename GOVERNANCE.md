# Governance

SunLit Security Libraries is currently maintained by Sherif Mansour as a
maintainer-led open-source project.

## Decision making

- Security fixes, vulnerability handling, releases, and trademark permissions
  are maintainer decisions.
- Non-trivial feature work should start with an issue or runbook under
  `docs/slo/`.
- Security-sensitive changes require tests and, when risk boundaries change, an
  update to `THREAT_MODEL.md` or the relevant developer guide.

## Releases

The crates are pre-1.0. Until the first stable release, API changes may occur,
but releases should include migration notes in [CHANGELOG.md](CHANGELOG.md).

Current maintainers are listed in [MAINTAINERS.md](MAINTAINERS.md).
